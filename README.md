## ☕STARBUCKS☕
실제 상용중인 스타벅스 사이트 카피

## ⏱️작업 기간
2024.10.16~2024.10.27

## 🚩 설계 목표
- 상용중인 사이트 카피
- 해당 사이트에 js를 사용하여 개발하는 방식으로 진행

## 📄 기능 구현
- 로그인/ 로그아웃
- 메안페이지 구현
- 고객의 소리, 문의글 조회, 자주하는 질문 보기
- 잘문중 매장 찾기 기능 및 음료 찾기 기능
